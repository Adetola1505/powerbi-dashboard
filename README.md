# Power BI Dashboard Project

## Overview
This project is a Power BI dashboard analyzing ticket system data.

## Key Insights
- Average ticket resolution time
- Ticket distribution by priority and status
- Trends in ticket volume

## Tools Used
- Power BI
- Data visualization
- Data cleaning (Power Query)
